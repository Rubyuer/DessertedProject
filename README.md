# 星巴图咖啡-商家版

应用说明: 该应用是管理"星巴图咖啡"的应用, 是对"星巴图咖啡"中商品的管理, 以及订单的处理的应用。

应用功能: 
      1、商品管理-对商品进行上架下架、修改、增添以及预览功能；   
      2、订单管理-对订单进行操作、订单流程的处理；     
      3、通过扫描订单二维码获取订单信息；      
      4、优惠券-查看商家发放的优惠券、新增发布优惠券；        
      5、收藏管理-查看用户收藏的统计；     
      6、评价管理-查看用户所提交的评价数据。
